# Админ-панель для Ticket Bot

Веб-интерфейс для управления ботом поиска билетов, просмотра активных трекингов и отправки сообщений пользователям.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Отредактируйте файл `.env` и настройте параметры админ-панели:

```env
# Настройки админ-панели
ADMIN_USERNAME=admin              # Имя пользователя для входа
ADMIN_PASSWORD=your_secure_password  # Пароль (смените в production!)
SECRET_KEY=your_random_secret_key    # Секретный ключ для сессий
FLASK_HOST=127.0.0.1                 # Хост для запуска
FLASK_PORT=5000                      # Порт веб-интерфейса
```

### 3. Запуск админ-панели

```bash
python admin_panel.py
```

Откройте браузер и перейдите по адресу: `http://127.0.0.1:5000`

## 🔐 Вход в систему

- **Логин:** `admin` (или значение из `ADMIN_USERNAME`)
- **Пароль:** `admin123` (или значение из `ADMIN_PASSWORD`)

⚠️ **Важно:** Смените пароль перед использованием в production!

## 📋 Функционал

### Дашборд
- Общая статистика (активные трекинги, пользователи, поиски)
- Статус мест по всем трекингам
- Популярные станции

### Управление трекингами
- Просмотр всех активных трекингов
- Детальная информация о каждом трекинге
- Удаление трекингов
- Включение/выключение heartbeat
- Настройка интервала heartbeat

### Пользователи
- Список всех пользователей бота
- Профиль пользователя с историей трекингов
- Быстрая отправка сообщения пользователю

### Отправка сообщений
- Отправка сообщений от имени бота любому пользователю
- Поддержка HTML форматирования
- Ограничение длины сообщения (4096 символов)
- Логирование всех отправленных сообщений

### Логи администратора
- История всех действий администраторов
- IP-адреса, время, детали операций
- Аудит безопасности

### API endpoints
- `GET /api/statistics` - Получить статистику
- `GET /api/trackings` - Получить все трекинги
- `DELETE /api/tracking/<id>` - Удалить трекинг

## ⚠️ Production рекомендации

1. **Смените пароли:**
   ```env
   ADMIN_PASSWORD=<сложный-пароль>
   SECRET_KEY=$(openssl rand -hex 32)
   ```

2. **Используйте HTTPS:**
   - Настройте reverse proxy (nginx)
   - Получите SSL сертификат (Let's Encrypt)

3. **Ограничьте доступ:**
   - Используйте firewall
   - Настройте whitelist IP
   - Рассмотрите basic auth на уровне nginx

4. **Мониторинг:**
   - Настройте алерты на ошибки
   - Мониторьте логи админ-панели
   - Отслеживайте подозрительную активность

## 📝 Примеры использования

### Отправка сообщения пользователю

1. Перейдите в раздел "Отправить сообщение"
2. Введите Chat ID пользователя
3. Напишите текст сообщения
4. При необходимости включите HTML форматирование
5. Нажмите "Отправить"

### Управление heartbeat трекинга

1. Перейдите в "Все трекинги"
2. Нажмите на иконку глаза для деталей
3. В разделе "Heartbeat настройки":
   - Pause/Start - включить/выключить
   - Измените интервал и сохраните

### Удаление трекинга

1. Найдите трекинг в списке
2. Нажмите кнопку удаления
3. Подтвердите действие

## 🆘 Troubleshooting

**Ошибка "Token must contain a colon"**
- Проверьте TELEGRAM_TOKEN в `.env`
- Токен должен быть в формате `123456789:AABBccDDeeFFggHHiiJJKKllMMnnOOpP`

**Не получается войти**
- Проверьте ADMIN_USERNAME и ADMIN_PASSWORD в `.env`
- Очистите cookies браузера

**База данных не найдена**
- Запустите основного бота для создания БД
- Или создайте директорию `data/` вручную
